Array Data Structure

An Array Data Structure is a collection of homogenious(same type) data items stored in contiguous(continuous) block of memory.

int arr[7]={233,14,23,19,-26,-254,38};

The above statement is creation of an integer type array of 7 elements.The addresses of the elements started from 50(these values may vary for each execution) ,ended at 74.That means the elements in that array are stored in a contiguous block of memory.Each element of an array is indexed starting from zero.In the above case it's 0-6

We can access any element of an array by using index of that element.
arr[0] will give 233
arr[1] will give 14
arr[2] will give 23
arr[3] will give 19
and so on..
one more way to access the elements of an array is by using pointers.We know that the name of the array is itself the address of first element in the array.

*(arr+0) will give 233
*(arr+1) will give 14
*(arr+2) will give 23
*(arr+3) will give 19
and so on..

1.Array Traversal

Traversing an array is printing all the values of that array.

#include<iostream>
using namespace std;
intmain(){
int i,arr[5]={23,54,87,96,1};
for(i=0,i<5;i++){
cout<<arr[i]<<" ";
}
return 0;
}
