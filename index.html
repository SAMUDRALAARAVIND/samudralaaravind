Pointers Introduction:

Pointers are variables that stores addresses of variables . i.e A Pointer to integer can store address of an int,Pointer to charecter can store address of a char variable.declaration of pointer variables are as follows:
int* ptr1;
char* ptr2;
float* ptr3;
In the above statements ptr1 can store address of an integer variable,ptr2 can store address of a char variable,ptr3 can store address of a float variable,

Note:Even though address of any variable is an integer But we can not store them in int type variable.

Reference(&) and Dereference(*) operators:

int x=200;
cout<<&x; //will give you the address of 'x'
In the above statement the variable x will be given 4 bytes of memory at some random location ,let it be 1000(address of x).To know the address of x we need a pointer to integer(x is of int type).
int* ptr;
now the variable ptr can hold the address of any data of type int.
ptr=&x;
To get the value at a location we use dereference(*) oprator.i.e int value=*(address);

we can also declare a pointer as int *y;,And we can declare both variable and pointer to variable in single statement as..int x,*y; Here x is int variable and y is pointer to int variable.

#include<iostream>
using namespace std;
int main(){
    int x=200;
    int* y=&x; //assigning address of x to pointer variable.
    cout<<"value at address "<<y<<" is: "*y<<endl;
    return 0;
}

Output:

value at address 0x6ffe14 is: 200

The number 0x6ffe14 is in hexadecimal form, in decimal form it is7339540.In your computer the value may not be the same.

Do You Know?

All variables and statements of a program will be stored in our computer's RAM,each address will represent a single byte of memory.Let's say if our computer has 2GB RAM ,then there will be 2*1024*1024*1024 memory addresses.After compleation of our Program all the data will be deleted except the data that is stored in the heap section(dynamically allocated memory).

Operations on Pointers:

We know that typically an int type variable requires 4 bytes of memory(depends on type of compiler).If address of a variable of int type is 0x6ffe14,when we add 1 to this address result will be 0x6ffe18 but not 0x6ffe15.

#include<iostream>
using namespace std;
int main(){
    int x=200;
    int* y=&x;
    cout<<" address of x is: "<<y<<endl;
    y++;
    cout<<"address next to x is: "<<y<<endl;
    return 0;
}

output:

address of x is: 0x6ffe08

address next to x is: 0x6ffe0c

0x6ffe0c-0x6ffe08=4
